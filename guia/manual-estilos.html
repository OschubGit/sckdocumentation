# Paletta

El proyecto usa diferentes paletas de color con una variedad de gamas de cada uno.

# Colores primarios

Disponemos de los primarios más una selección sck colors que cuentan con opacidad.

// Palette primary colors
$primary-color: #20c0c5;
$primary-black: #06070d;
$dark: #0e0e0e;
// End Palette primary colors

# Colores SCK

Estos colores son los originales de scenikus.

// sck colors
$secondary-color: #112B31;
$color-secondary: #d12465;
$green-color: #62b340;
$yellow-color: #ebb128;
$blue-color: #4b90cd;
// end sck colors

# Color background y contrast background

$primary-background: #06070d;

// Contrast background

# Font Family

El nuevo proyecto de Scenikus usa Roboto. Esta tipografía es la que viene por defecto al instalar Material Ui.

# Importación

La tipografía ya está importada en el proyecto. En cualquier caso podemos cambiarla o agregar más pegando el link de importación en el head del archivo index.html

<link href="https://fonts.googleapis.com/css?family=Roboto" rel="stylesheet" type="text/css">

# Uso Css

Usaremos variables para importar las tipografías y usarlas a nivel global. Encontramos la carpeta en SCSS > utilities > _variables.scss

$font-title: 'Montserrat', sans-serif;
$font-text: 'Open Sans', sans-serif;

Luego podremos llamar estas variables en los archivos Scss:

font-family: $font-title;

# Color para tipografías

También podemos especificar colores para las tipografías. Usamos colores de contraste en las tipografías para resaltarlas del fondo $primary-background

$light: #fff;
$medium: #dadada;
$contrastTextv1: #253746;
$contrastTextv2: #192735;
$contrastTextv3: #06070D;

# Font size

En _variables.scss especificamos tamibén el tamaño de las tipografías. Material Ui tiene el tamaño por defecto mediante:

<Typography variant="[variante]"></Tipography>
